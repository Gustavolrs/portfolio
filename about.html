Sobre mim

Iniciei minha jornada no Ensino Médio em uma escola técnica de informática, onde desenvolvi uma sólida base em tecnologia. Após isso, trabalhei por um ano como auxiliar administrativo, onde adquiri experiência em software de escritório, processamento de dados e controle de estoque.

Atualmente, estou cursando Análise e Desenvolvimento de Sistemas, dedicando-me a cursos para expandir meu conhecimento. Já conquistei certificações em algoritmos, Java, bancos de dados (MySQL), JavaScript, HTML, CSS, e Angular com Spring Boot.

Minha experiência prática inclui o desenvolvimento de aplicações web usando Angular e Spring Boot, a criação de interfaces de usuário dinâmicas com HTML, CSS e JavaScript, e a gestão de bancos de dados com MySQL. Tenho aplicado esses conhecimentos em projetos pessoais e colaborativos, buscando sempre aprimorar minhas habilidades e aprender novas tecnologias.

Estou constantemente buscando novas oportunidades para aplicar e aprimorar minhas habilidades em desenvolvimento de software e contribuir para projetos inovadores.
